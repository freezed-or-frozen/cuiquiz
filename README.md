# CuiQuiz


## 1 - Introduction
Web app quiz game like **Kahoot!** to use in a classroom


## 2 - Installation
Install all dependencies with **npm**
```
npm ci
```

Start **node.js** server with
```
node ./index.js
```

In a web browser : 
- (http://127.0.0.1:3000/teacher) for teacher
- (http://127.0.0.1:3000/student) for student
